Debian Wheezy - Setting up Oh-My_Zsh

Iam currently running Google Cloud Services just for testing and learning purposes which I got for free. Today I deployed Lamp Stack on Linux (Apache, MySQL, and PHP)

Iam thinking of Hosting a Wordpress blog on that server maybe later in the next few months .

Since the server runs Debian Wheezy and ofcourse no GUI interface means having to SSH into that server all the time.

In comes Oh-My-Zsh! I simply love Zsh compared to Bash terminal! I have Zsh running on my Ubuntu and Mac Terminals and I thought it would be a walk in the park to have it up on this server which was not so straight forward really

This is what got me saying Oh My Zsh

Installing Zsh on Debian 


  $ sudo apt-get update -y 
  $ sudo apt-get install -y zsh

Trying to run Oh-My-Zsh from Robby Russel's Git threw in some errors that Debian didnt have any Git Installed

I Installed Git 

 $ sudo apt-get install libcurl4-gnutls-dev libexpat1-dev gettext libz-dev libssl-dev build-essential
 $ wget https://github.com/git/git/archive/v1.9.4.tar.gz

After it downloads, untar the file and switch into that directory: 


$ tar -zxf v1.9.4.tar.gz
$ cd git-1.9.4
$ make prefix=/usr/local all

Setting up Git, The Git config file does not exist and I had to create one 


 $ vim ~/.gitconfig
 [user]
 name = "my git username"
 email = "my git email address "

This shows the configurations done 


git config --list
