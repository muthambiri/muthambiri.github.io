Compute Engine

I just deployed a Virtual Machine Instance with'Click to Deploy' image based on Lamp Stack (Apache, PHP & MySQL) In coming months I will move this blog to that server.

The VM instances can also be created manually by selecting, Location (US or Europe) & resources like Boot Disk Size , Firewall (allow HTTP & HTTPS Traffic) Images (Debian, OpenSuse, RedHat Enterprise, CoreOS, Suse)
